# 350mil_param_GPT

A modern GPT implementation featuring **Exclusive Self-Attention (XSA)**, **ReLU² activation**, a **Muon + AdamW hybrid optimizer**, and **DeepSeek-style scaled residual pathways**. The model is designed with a strong focus on **efficiency, stability, and performance at scale**, incorporating several improvements inspired by recent large language model architectures.

The project maintains a **clean separation between model and training components**, resulting in a modular and readable PyTorch codebase. It is trained on **NVIDIA’s ClimbMix dataset**, with an emphasis on achieving strong learning dynamics while keeping computational cost under control.

Overall, this work explores **practical enhancements beyond standard GPT designs**, combining modern ideas in attention mechanisms, optimization strategies, and residual scaling into a single scalable system.
